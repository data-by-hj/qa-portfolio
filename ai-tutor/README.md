# 교내 AI 조교 프로그램 QA

## 프로젝트 개요
교수자 대상 LLM 기반 문제 생성 AI 조교 서비스의 품질 검증 프로젝트입니다.
기능 테스트, 보안 테스트, UI/UX 테스트를 직접 수행하였습니다.

- **기간**: 2026.02 ~ ing
- **역할**: QA 단독 수행 (테스트 케이스 → 수행 → 버그 리포트 → 회귀 테스트)
- **테스트 규모**: 1차 99개 + 2차 228개, 총 327개 테스트 케이스 수행

## 주요 발견 버그

### 1차 테스트 (2026.02)

#### 🔴 Critical (보안)
- **XSS 취약점**: LLM 대화 입력창에 `<script>alert(1)</script>` 입력 시 alert 팝업 실행
- **API 토큰 평문 노출**: 개발자 도구 Network 탭의 Request Headers에서 `x-access-token` 평문 노출
- **로그아웃 후 토큰 재사용**: 로그아웃 후 복사한 토큰으로 API 재요청 시 정상 응답 반환 (401 미처리)

#### 🟠 High
- **다중 파일 처리 오류**: PDF 여러 개 첨부 후 요약 요청 시 답변 생성 단계에서 오류 발생 및 자동 로그아웃
- **시험지 저장 DB 오류**: 보관함 시험지 불러온 후 문제 추가 → 덮어쓰기 저장 시 `question_id` NOT NULL 제약 조건 위반 오류
- **교육 스타일 / 연구 프로파일 저장 미작동**: 변경 후 화면 이탈 시 이전 값으로 초기화
- **스트리밍 롤백 현상**: 답변 생성 중 텍스트 출력 후 다시 "답변 생성 중" 상태로 복귀
- **시험지 채점 목록 미노출**: 기출문제가 존재함에도 채점 탭 드롭다운에 시험지가 표시되지 않음
- **LLM 컨텍스트 미유지**: 조건을 나눠 입력 시 이전 발화 내용이 반영되지 않음. 일괄 입력 시 정상 동작
- **보관함 시험지 문제 누락**: 불러온 시험지에 기존 저장 문제가 로드되지 않음

#### 🟡 Medium
- **아이디 저장 미작동**: 체크박스 선택 후 로그아웃 시 아이디가 저장되지 않음
- **LLM 파일 목록 불일치**: 파일 관리에 업로드된 목록과 LLM이 반환하는 파일 목록이 불일치
- **문항 순서 불일치**: 보관함 / 기출문제 / PDF / 미리보기 간 문항 순서가 각각 다르게 출력
- **피드형 보기 다중 삭제 불가**: 리스트형에는 체크박스 있으나 피드형에는 선택 수단 없음
- **LLM 모델 선택 초기화**: 답변 생성 후 매번 GPT로 모델 선택이 초기화됨
- **로그아웃 후 뒤로가기 접근 가능**: 브라우저 뒤로가기 시 인증 없이 이전 페이지 접근 가능

#### 🔵 Low
- **화면 축소 시 레이아웃 깨짐**: 브라우저 화면 축소 시 가로 스크롤 미제공, 일부 콘텐츠 확인 불가
- **입력 텍스트 미표시**: 그룹명, 문제 주제 등 여러 입력창에서 입력 텍스트가 보이지 않고 드래그 시에만 확인 가능

### 2차 테스트 (2026.03 ~ 2026.04)

> 1차에서 발견된 버그의 수정 여부 검증 + 신규 테스트 케이스 추가 수행

#### 🔴 Critical (보안)
- **XSS 취약점 미수정**: 1차와 동일하게 alert 팝업 실행 확인 — 끝까지 미수정
- **API 토큰 평문 노출 유지**: `x-access-token` 평문 노출 지속 확인
- **로그아웃 후 토큰 재사용 유지**: 로그아웃 후에도 기존 토큰으로 API 정상 응답 반환

#### 🟠 High
- **소스문서 파일 참조 오류**: 소스문서 삭제 후에도 LLM이 해당 파일 내용을 참조하여 답변
- **저장 버튼 UI 누락**: 교육 스타일 프로파일링 화면에 저장 버튼 자체가 없어 변경 불가
- **시험지 점수 초기화**: 보관함에서 불러온 시험지의 문항 점수가 0점으로 초기화, 총점 계산 오류 동반
- **기출 시험지 총점 동기화 오류**: 한 시험지 총점 변경 시 다른 기출 시험지 총점도 동일하게 변경됨
- **기출 시험지 원본 데이터 변경**: 기출 시험지 이름 변경 및 문제 수정 시 원본 시험지 데이터도 함께 변경됨
- **시험지 채점 자동 로그아웃**: 학생 시험지 업로드 시 세션 만료 오류 및 자동 로그아웃 발생
- **내부 시스템 프롬프트 노출**: 내부 정책 질의 시 시스템 프롬프트 일부 노출 (1차 발견, 2차 수정 확인)

#### 🟡 Medium
- **아이디 저장 미수정**: 1차·2차·3차 테스트 모두에서 동일하게 재현, 끝까지 미수정
- **답변 생성 중 입력창 잠금 해제**: 다른 탭 전환 후 복귀 시 비활성화 상태가 풀려 추가 입력 가능
- **피드형 보기 다중 삭제 불가 유지**: 1차와 동일하게 미수정 확인
- **대화 내역 최초 클릭 시 새 창**: 대화 항목 첫 클릭 시 기존 대화 대신 새 대화창이 열리고 두 번째 클릭부터 정상 동작
- **문제 생성 수량 불일치**: 요청한 문항 수와 다른 수량이 생성되는 현상 반복 확인
- **상충 조건 처리 미흡**: 상충되는 조건 입력 시 충돌 인식 없이 임의로 문제 생성
- **강의자료 수정 후 문제 생성 오류**: 강의자료 섹션 수정 후 해당 내용 기반 문제 생성 요청 시 오류 발생

#### 🔵 Low
- **코드 블록 가독성**: 코드 블록 글씨색과 배경색이 유사하여 가독성 확보 불가
- **시험지 이름 안내 문구 미노출**: 특수문자/공백 입력 시 안내 문구가 PDF 버튼 뒤에 가려짐
- **화면 확대 시 레이아웃 깨짐**: Ctrl+휠 확대 시 가로 스크롤 미제공
- **소스문서 페이지당 표시 수 감소**: 기존 대비 1페이지당 2개만 표시
- **내부 HTML 코드 노출**: 소스문서 참조 답변 시 `hljs-punctuation` 등 내부 스타일 태그가 그대로 출력됨
- **동영상 업로드 상태 미표시**: 강의 영상 업로드 후 업로드 완료 여부를 목록에서 확인 불가
