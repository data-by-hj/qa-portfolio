## 📂 프로젝트 경험

**산후우울증 조기 발견 플랫폼 '맘포유' 기획** 2024  
공공 빅데이터 활용 캡스톤 경진대회 **장려상** | 강원지역혁신플랫폼 정밀의료사업단 | 팀 프로젝트 | 역할 : 회의 주도 및 발표

- 문제 정의: 산후우울증의 낮은 조기 발견율 → 커뮤니티 데이터 기반 조기 감지 플랫폼 기획
- 데이터 설계: 국립국어원 감성사전으로 커뮤니티 게시글 감성 분석, 보건복지부 정신건강 데이터·에딘버러 자가진단 데이터 등 공공데이터 결합
- 솔루션: 익명 커뮤니티 + 우울증 자동 진단 + 전문가 상담 연결 기능으로 구성된 통합 플랫폼 설계

**Phone Scam Detection 글로벌 프로젝트** 2024  
강원지역혁신플랫폼 × 하노이과학기술대학교(정보통신대학) 협업 | 팀 프로젝트 | 파이프 라인 설계

- 문제 정의: 보이스피싱이 한·베트남 공통 사회 문제임을 확인 (2023년 베트남 피해액 약 8,000~10,000억 동, 73%가 전화·문자 경로)
- 데이터: 캐글에서 한국어 통화 텍스트 및 영어 스캠 통화 데이터 수집
- 파이프라인 설계: 음성 → Google Speech-to-Text → 텍스트 수치화 → SVM 분류 → 고령자 실시간 경고 알림
- 비즈니스 모델 수립: 통신사·정부기관 연계 및 구독형 수익 구조 설계

**'짱구는 못 말려' 캐릭터 음성 분류 모델** 2025  
SW 동아리 우수작품 전시 **동상** | Python · scikit-learn · Gradio | 1인 프로젝트

- 데이터 수집: 애니메이션 mp3에서 12개 캐릭터 음성 구간 수동 추출, 성별/캐릭터 폴더 구조로 체계화
- 전처리 파이프라인: 16kHz 리샘플링 → 하이패스 필터 → 무음 구간 제거 → RMS 정규화
- 특징 공학: MFCC 기반 약 50차원 특징 벡터 구성, 고음 캐릭터 구분력 강화
- 모델링: RBF 커널 SVM → 극소 데이터 환경에서 정확도 70% 달성
- Gradio UI: 마이크 실시간 입력 + 파일 업로드 지원, TOP5 유사 캐릭터 확률 그래프 시각화
